What is HTML?

HTML stands for Hypertext Markup Language, it's a markup language developed by sir Tim-Berners lee in 1993 to develop web pages.

HTML consists of elements that describes different types of content: Headings, Paragraohs, Links, Images, Lists, tables etc.
